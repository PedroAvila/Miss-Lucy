Terminantes: ANDO - ENDO    I am cooking

ING => Verbos

Reglas
======

# Agregar ING
Buy => Buying
Go => Going
Read => Reading
I am reading a book 

# Eliminar la e y agregar ING
Belive => Beliving
Practice => Practicing

# Monosilábico - Terminación CVC(Consonante-Vocal-Consonante) Duplicar + ING
Jog => Trotar
cvc
Jogging => Trotando
Chop => Cortar
c vc
Chopping => Cortando

# Excepciones
W - X - Y - Z
Draw => Drawing
Play => Playing
